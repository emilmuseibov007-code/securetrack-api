# SecureTrack API

Учебный проект на Python, FastAPI и PostgreSQL для учета сетевых устройств и инцидентов информационной безопасности.

## Описание проекта

SecureTrack API — это небольшой backend-сервис, который позволяет:

- добавлять сетевые устройства
- просматривать список устройств
- добавлять инциденты информационной безопасности
- просматривать список инцидентов
- получать инциденты, относящиеся к конкретному устройству

Проект создан в учебных целях для демонстрации работы с API, базой данных PostgreSQL и связями между таблицами.

## Используемые технологии

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Uvicorn
- Pydantic

## Структура базы данных

### Таблица `devices`
Хранит информацию об устройствах:

- `id` — идентификатор
- `hostname` — имя устройства
- `ip_address` — IP-адрес
- `os` — операционная система
- `location` — местоположение
- `owner` — владелец
- `created_at` — дата создания записи

### Таблица `incidents`
Хранит информацию об инцидентах:

- `id` — идентификатор
- `title` — название инцидента
- `description` — описание
- `severity` — уровень серьезности
- `status` — статус
- `discovered_at` — дата создания записи
- `device_id` — ссылка на устройство

## Связь между таблицами

Одно устройство может иметь несколько инцидентов.

Связь реализована через внешний ключ:
- `incidents.device_id -> devices.id`

## Реализованные маршруты API

### Устройства
- `GET /` — проверка работы API
- `GET /devices` — получить список устройств
- `POST /devices` — добавить новое устройство

### Инциденты
- `GET /incidents` — получить список инцидентов
- `POST /incidents` — добавить новый инцидент
- `GET /devices/{device_id}/incidents` — получить инциденты выбранного устройства

## Запуск проекта

Для запуска проекта необходимо открыть терминал в папке проекта, активировать виртуальное окружение, установить зависимости и запустить сервер FastAPI.

1. Активировать виртуальное окружение:

```bash
source /Users/sq/pyth/venv/bin/activate

2. Установить зависимости:

```bash
pip install -r requirements.txt

3. Запустить сервер:

```bash
uvicorn app.main:app --reload --port 8001

4. Открыть Swagger UI в браузере:
http://127.0.0.1:8001/docs

5. Корневой маршрут API для проверки работы сервера:
http://127.0.0.1:8001

9. Результат
